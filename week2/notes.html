All About Functions

How do you define a function?

There are multiple ways to define a function:

  1. function declarations: function hello(){console.log('Hello World!')}
  2. anonymous functions: function(){ console.log('Goodbye World!')}
  3. function literal: const goodbye = function(){ console.log('Goodbye World!'); };
  4. function expressions: const goodbye = function bye(){ console.log('Goodbye World!'); };
  5. Function() constructor: const hi = new Function('console.log("Hi World!");'); ( This way is NOT recommended)

What's an arrow function?

Succinct anonymous functions. The parameter comes before the =>. Examples:

  1. const square = x => x*x;
  2. const add = (x,y) => x + y;
  3. No parameters is: const hello = () => alert('Hello World!');
